Zoznamy

Cielom dnešneho cvičenia je vytvoriť stránku,ktorá vyzera rovnako ako táto.

Všetky potrebne informacie nájdete tu.

Vytvorenie nezoradeneho zoznamu

  • Vytvorte novy HTML dokument s nazvom napr. list.html a inicializujete ho
  • Nastavite velkost pisma na 20px
  • Vytvorite header Zoznamy
  • Vytvorite paragrafy s obsahom zhodnym s prikladov
  • Vytvorite header pre nezoradeny zoznam
  • Vytvorite zoznam pomocou tagu <ul>
  • vložite obsah zhodny s prikladom pomocou tagu <ul>

Vytvorenie zoradeneho zoznamu

  1. vyvorite header pre zoradeny zoznam
  2. vyvorite zoznam pomocou tagu <ol>
  3. vložite obsah zhodny pomocou tagu <li>

Vyvorenie zoznamu s pojmami a ich defniciami

zoradeny zoznam
